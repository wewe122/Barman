# Barman

השחקן צריך להכין את כל המשקאות שהלקוחות מבקשים בזמן מוקצב ולאסוף כמות גבוהה של כסף בתמורה לשירות.
השחקן יופיע בתוך בר עם כל סוגי המשקאות מונחים לפניו ולאט לאט יגיעו לקוחות ויבקשו שירות.
